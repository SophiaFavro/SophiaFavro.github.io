Overview & Usage

The extension I created is called the Study Buddy! It allows you to have a little pet icon pinned where you can create a to-do list and recieve encouraging words from your buddy. The best way to run this code is to download the file, and then unpack the file in the extensions menu of your browser and pin it so it shows in the taskbar.
Here is a link to the repository:Extension Code

Development Process

I knew I wanted there to be a popup, so I used the "hello world" example and customized it for my project. I did this by adding my own icon and adusting the styling, manifest, html, and JavaScript console log.

After the popup was complete it was time to start working on the to-do list. I started out by creating a button to add tasks to the list. As well as creating an input for the user to be able to type in their task.

I then needed to create an event handler that would be able to listen for when the "add task" button was clicked in order to take the input, and add it to the list in the popup. To do this it was time to start working in my js file. I started by creating variables for the button, user input, and list. Then creating a function for when the "add task" button was clicked that would add the input to the list and give the list item a "done" button for when the task was complete. Below is the list filled out with the tasks and you can see each has a "done" button that when clicked removes the item.

Issue Deep-Dive

Once I added the button for adding a task and input I ran into an issue having the items added to the list and visible in the popup. I was able to type in a task and click the "add task" button, but it would just dissapear instead of being added below. To go about fixing this I decided to add "console.log" within my event hadler for clicking on the button that would log the input(what was typed in by user).

As you can see the console was logging it, so I knew that my event handler was working. I thought maybe the length had something to do with why it wasn't appearing, but when I tried just "add" it still wasn't appearing. Due to this I decided to look and see if there was an issue in my html or css files instead of my js file. I looked line by line to make sure that everything was matching and there were no syntax issues. While doing this in the html I saw my empty list ("ul") tags and thought the list is empty and it should be adding the list item. In this moment I realized that I may have appended the listButton and listText to the listItem, but had not appended the listItem to my list. Meaning that my main parent node for the to-do list had no children, but the child was still being made. Everything was working and logging and the issue was that they just needed to be connected. Once I added list.appendChild(listItem); the items were apearing in the popup and being added to the to-do list.
