Overview & Usage

The extension I created is called the Study Buddy! It allows you to have a little pet icon pinned
where you can create a to-do list and recieve encouraging words from your buddy.
The best way to run this code is to unpack the file in the extensions menu of your browser and pin it so it shows in the taskbar.

Development Process

I knew I wanted there to be a popup, so I used the "hello world" example and customized it for my project.
I did this by adding my own icon and adusting the styling, manifest, html, and JavaScript console log.

After the popup was complete it was time to start working on the to-do list.
