Project Description

A web app that allows users to create pictures/draw and download their picture.

Interface Design

Versions Overview

Prototype version: Set colors to choose from and only able to stamp on shapes to draw, and just able to download it.
Final version: Color picker to choose colors, able to draw with shapes and lines, and download informs users of when its complete/keeps the user informed throughout.

Development Plan

  • Setup local development server/environment and add necessary packages (p5)
  • Create color and shape pallet used to draw
  • Add functionality for color/shape selection
  • Add functionality for adding color/shape to blank canvas
  • Add functionality that allows user to download drawing
  • Use rollup and deploy version to gh pages - Milestone finished prototype version
  • Add color picker to pallet selection choices
  • Add functionality for drawing lines/freeform lines
  • Add functionality for informing user of downloading steps
  • Use rollup and deploy version to gh pages - Milestone finished final version

Known Unknowns

  • Allowing users to download their picture
  • Adding a color picker
  • Have stroke size change correlate to sliding bar

Stretch Goals

  • Create stamps that user can use on their canvas
  • Able to add animations to their drawings
  • Adding grid that can be made visible/invisible
